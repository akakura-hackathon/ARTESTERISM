# Art Partner - 4th Google Cloud Hackathon Dev

Google Cloudハッカソンの開発用プロジェクト

## 概要

美術館向け多言語対応アートガイドアプリケーション

## 技術スタック

- React + Vite
- Firebase (Authentication, Hosting, Cloud Storage)
- i18next (多言語対応)
- Material-UI

## 機能

- 多言語サポート（日本語、英語、中国語、韓国語、スペイン語、フランス語、ロシア語）
- 美術館・作品情報の管理
- 音声ガイド機能
- ユーザー認証
