# ARTESTERISM

第4回 Agentic AI Hackathon with Google Cloud の提出用

## 概要

ARTESTERISMは、美術館と来場者をAIとデータでつなぎ、鑑賞体験と美術館運営の双方を支援するプラットフォームです。Geminiによるレベル別・多言語の作品解説生成と、BigQueryのエンベディングを活用した作品レコメンドにより、来場者には自分の興味度に合った鑑賞体験を提供します。同時に、作品単位のフィードバックデータを可視化・分析することで、美術館の展示改善や経営判断を支援します。大規模美術館から中小規模美術館への回遊を促進し、日本の美術館体験の持続的な発展を目指します。

## 機能

- ユーザ認証
- 多言語サポート（日本語、英語、中国語、韓国語、スペイン語、フランス語、ロシア語）
- ユーザの興味に基づいた音声の提供
- ユーザー(来場者)の絵画作品に対するフィードバック入力
- 収集されたユーザーの好みをベースに次に見るべきおすすめ作品を提示
- ユーザー(美術館職員)向けに、分析のために各絵画作品に対するユーザー評価を提供
